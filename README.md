# EASYIMOB - MOBILE

Projeto Extensionista para a disciplina "Topicos de Big Data em Python"

**Objetivo:** Criar um aplicação desktop usando Python e aplicando conceitos de Big Data

**Foco:** Um gerenciador capaz de armazenar dados dos clientes e dos imoveis, para corretores imobiliarios. Alem permitir a busca de imoveis em portais de anuncios, conforme o perfil do cliente.

**Funções:**
* Manter cadastro de Clientes e Imoveis
* Permitir a busca de anuncios no portal OLX(escolha do parceiro externo)
* Automatizar a busca de anuncios conforme o perfil de preferencias do cliente.
* Agendamento de buscar periodicas, alem de enviar notificações via email dos anuncios encontrados.
